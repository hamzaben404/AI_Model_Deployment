Welcome to the AI Model Deployment Application

This application is designed to demonstrate how to deploy various AI models using Flask and Docker. Below, you will find an overview of the application's endpoints, how it works, and step-by-step instructions to deploy it automatically.

Available Endpoints

  • /predict: Performs image classification using the VGG16 model. Upload an image file to get predictions of objects in the image.
  • /regpredict: Provides predictions using a regression model. Send numerical features in JSON format to receive predictions.
  • /textgen: Generates text using the Hugging Face GPT-2 model. Provide a text prompt in JSON format to receive generated text.
  • /home: Displays this documentation page.

Application Architecture

The application is structured as follows:

  • Flask: A lightweight Python framework used to expose AI models through RESTful APIs.
  • VGG16: A pre-trained deep learning model for image classification.
  • Regression Model: A Pickle-based machine learning model trained for regression tasks.
  • Hugging Face: A text generation model (GPT-2) for natural language tasks.
  • Docker: A containerization platform to package and deploy the application.

Deployment Instructions

Follow these steps to deploy the application automatically:

1. Clone the Repository

git clone https://github.com/hamzaben404/AI_Model_Deployment.git
cd AI_Model_Deployment

2. Build and Run the Docker Container

Ensure Docker is installed on your machine, then run the following commands:

docker-compose build
docker-compose up

3. Access the Application

Once the Docker container is running, you can access the application at:

  • Base URL: http://127.0.0.1:5000
  • Access specific endpoints such as /predict, /regpredict, or /textgen using tools like Postman or cURL.

Testing the Application

Here are examples of how to test the endpoints:

  • Image Classification:
    • curl -X POST -F "file=@path_to_image.jpg" http://127.0.0.1:5000/predict
  • Regression Model:
    • curl -X POST -H "Content-Type: application/json" \ -d '{"features": [1.5]}' \ http://127.0.0.1:5000/regpredict
  • Text Generation:
    • curl -X POST -H "Content-Type: application/json" \ -d '{"prompt": "Once upon a time"}' \ http://127.0.0.1:5000/textgen
